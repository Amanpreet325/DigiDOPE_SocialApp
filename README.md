- Project Name: DigiDope
- Technologies Used:
  - Node.js
  - Express
  - Multer
  - Passport.js
  - MongoDB
- Features:
  - User Registration and Login
  - Profile Management
  - Posting Functionality
  - Image Upload for Profile Pictures and Posts
  - Authentication with Passport.js
- Functionality Overview:
  - Users can register and log in securely.
  - Upon logging in, users can access their profile page to view and manage their posts.
  - Posting functionality allows users to create posts with captions and descriptions.
  - Users can upload images for both profile pictures and posts.
- Database Management:
  - MongoDB is utilized to store user data, including profiles and posts.
- User Experience:
  - Intuitive user interface for easy navigation between different screens.
  - Profile page provides a comprehensive view of user posts and allows for profile picture updates.
- Overall Purpose:
  - DigiDope aims to provide a complete social media experience, facilitating user engagement and interaction in a secure and user-friendly environment.
